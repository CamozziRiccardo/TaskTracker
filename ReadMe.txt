Il programma è un gestore di task a livello aziendale, con un sistema di utenti che permette di salvare il proprio utente e assegnarli sia l'azienda  a cui appartiene che la posizione in cui ti ritrovi nella suddetta azienda, una volta eseguito l'accesso si potrà accedere all'area in cui vengono gestite le task. In base alla posizione in cui ti ritrovi potranno essere eseguite funzioni differenti.
Il task manager prevede visualizzazione dell task, l'aggiunta di task, spuntare le task completate ed eliminazione delle task completate.
In base alla posizione in cui ci si trova all'interno dell'azienda le funzioni varieranno o verranno espanse.
Il programma dispone di una interfaccia grafica semplice ed intuitiva sviluppata con i windows form.
